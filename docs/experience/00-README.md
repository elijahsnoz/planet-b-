# Planet B — Experience Redesign

## From Website to World

> *I don't want visitors to say "that was a beautiful website." I want them to say "I've never experienced anything like that."*

This folder reimagines the **felt experience** of Planet B. The backend, schema, and admin already make it an institution that can last a century. This is about the first ninety seconds — and the hour after — that make a stranger fall in love with the only planet we have.

**Design only.** Per the brief, nothing here is implemented until approved. It builds on, and never contradicts, the [Founding Principles](../00-PRINCIPLES.md), the [Manifesto](../02-movement-manifesto.md), and the existing [Design System](../05-design-system.md) / [Motion Principles](../06-motion-principles.md) — it deepens them from "premium website" to "another world."

## The reframe — vocabulary we now design with
We stop designing *pages* and start designing *passages*. The old words are retired:

| ✗ Website thinking | ✓ World thinking |
|---|---|
| Hero | **Threshold** |
| About | **Memory** |
| Features / Gallery | **Discovery** |
| Content / Archive page | **Archive** |
| User flow | **Journey** |
| CTA | **Invitation** |
| Footer | **Legacy** |

## The four questions every section must pass
Before anything ships, it answers yes to all four:
1. Would this make someone **stop scrolling**?
2. Would a **museum** be proud of it?
3. Would a **child remember** it?
4. Would a **historian preserve** it?

If no → redesign.

## What we are NOT doing
No gimmicks, no parallax-for-its-own-sake, no autoplay sound, no "tech-startup" gloss, no performance-killing spectacle. Every gram of motion, sound, and weight **earns its place** or is removed. Timeless over trendy. Calm over loud. Meaning over flash. And all of it degrades gracefully to a quiet, readable, accessible experience (`prefers-reduced-motion`, no-JS, slow networks in Abuja).
